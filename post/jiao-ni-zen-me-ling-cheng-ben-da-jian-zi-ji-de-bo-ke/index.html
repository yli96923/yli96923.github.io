教你怎么零成本搭建自己的博客

2024-10-19 2 min read # Gridea

跟着这个教程来，你就能搭建一个我这样的博客！本次使用的工具是 Gridea 和 Github Pages，Github 上不去的话可能需要梯子辅助。

首先找到 Gridea 在 Github 上面的项目，不是 Gridea 自己的网站（那个收费）。也可以点击这个网址直接跳转。在页面的右边找到「Releases」选项，选择最新的一个下载你的操作系统对应的版本然后安装就好了。

安装好了之后，打开 Gridea，在页面左边找到「远程」，然后打开 Github，创建一个仓库，名称叫做「你的用户名.github.io」，然后不用更改其他选项，直接创建就好。然后创建一个 Token，可以直接点击这个链接跳转，「Note」可以填写一个好记得名字，例如「Gridea」，「Expiration」是这个 Token 的有效期，按需填写，最后在「Select scopes」选项里面勾选上「Repo」以及下面的所有选项，然后创建 Token，别忘了复制下来，因为关闭页面之后就再也找不到这个 Token 了！

打开 Gridea，选择左边的「远程」选项，根据你的信息配置就好了，大家可以以参考我的配置，不过「域名」「仓库名称」和「仓库用户名」都要填你自己的，「令牌」就是刚刚复制的 Token。随后点击保存和同步，等一会儿再登上你的网站就可以看到内容了！然后再把想要发布的内容在 Gridea 里面发布，就会直接同步到你的网站上，排版什么的都已经完成好了。
